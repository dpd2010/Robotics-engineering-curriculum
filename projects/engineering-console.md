# Engineering Console

## Problem Statement

Create a simple terminal dashboard that summarises simulated engineering measurements and the next action.

## Objectives

- Display project status clearly.
- Summarise simulated sensor values.
- Record evidence for the engineering notebook.

## Difficulty

Foundation

## Estimated Build Time

2 hours

## Components

- Python
- Terminal
- Markdown notebook

## Circuit

No physical circuit required for the first version.

## Architecture

- Measurement input layer
- Calculation layer
- Terminal output layer
- Notebook evidence layer

## Build Stages

1. Plan output sections.
2. Build calculation functions.
3. Print measurement summaries.
4. Capture output.
5. Document lessons learned.

## Testing

- Run with normal values.
- Run with extreme values.
- Check output labels and units.

## Lessons Learned

- Readable output makes debugging faster.
- Small functions are easier to test.

## Future Improvements

- Load readings from CSV.
- Add live hardware data.
