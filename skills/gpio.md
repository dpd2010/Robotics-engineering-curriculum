# GPIO

## Purpose

GPIO lets software read and control physical pins on a microcontroller or single-board computer.

## Core Ideas

- Pins can be configured as inputs or outputs.
- Pull-up and pull-down resistors control default input states.
- GPIO code should include safe startup and shutdown states.
