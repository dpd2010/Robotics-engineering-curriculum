# Raspberry Pi Zero

## Purpose

The Raspberry Pi Zero is a small Linux computer useful for robotics control, logging, networking, and higher-level software experiments.

## How It Works

It runs Raspberry Pi OS and can execute Python, host scripts, communicate over Wi-Fi, and interact with hardware through GPIO.
