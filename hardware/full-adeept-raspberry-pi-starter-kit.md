# Full Adeept Raspberry Pi Starter Kit

## Purpose

The Adeept starter kit provides common electronics components for robotics and embedded systems practice.

## How It Works

The kit combines sensors, actuators, resistors, wires, and display components that can be connected to Raspberry Pi or microcontroller projects.
